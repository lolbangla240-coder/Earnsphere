# 🌐 EarnSphere – GitHub Upload Guide

## 📁 এই Folder-এর Files:

| File | কাজ |
|------|-----|
| `index.html` | মূল App |
| `styles.css` | সব Design/CSS |

| `app.js` | সব App Logic & Data |
| `sw.js` | Offline Support (PWA) |

| `icon-512.png` | App Icon (বড়) |

---

## 🚀 GitHub-এ Upload করার Steps:

### Step 1 — GitHub-এ যান
👉 https://github.com এ Login করুন

### Step 2 — Repository খুলুন
আপনার EarnSphere repo-তে যান

### Step 3 — পুরানো files Delete করুন
- আগের `index.html` বা যেকোনো পুরানো file থাকলে delete করুন

### Step 4 — নতুন Files Upload করুন
- "Add file" → "Upload files" click করুন
- এই folder-এর **সব ৮টা file** একসাথে drag & drop করুন
- Commit message লিখুন: `Add PWA support and split files`
- "Commit changes" click করুন ✅

### Step 5 — GitHub Pages চেক করুন
- Settings → Pages → আপনার site live হবে

---

## 📱 Mobile-এ App Install করবেন কীভাবে?

**Android:**
Chrome-এ site open করুন → নিচে "Add to Home Screen" → Install ✅

**iPhone:**
Safari-এ site open করুন → Share button → "Add to Home Screen" ✅
